# LaunchOS Skill

> macOS 菜单栏应用启动器 [LaunchOS](https://apps.apple.com/app/launchos) 的智能分组管理工具。AI 分析你的应用习惯，生成个性化分组方案，无需手动拖拽。

## 能做什么

对 AI 说几句话，就能完成：

- 🧠 **智能整理** — AI 分析你的应用分布，推理用户画像，生成个性化分组方案
- 📁 **管理文件夹** — 创建、合并、重命名、删除分组
- 🔍 **查找死链接** — 检测已卸载但条目残留的应用
- 🆕 **发现新应用** — 扫描 /Applications 找出未收录的应用
- 💾 **备份恢复** — 操作前自动备份，出错一键回滚
- ⚙️ **调整外观** — 修改 LaunchOS 的列数、快捷键、背景等偏好

## 安装

**Claude Code 用户：**
```bash
/plugin marketplace add Jonykit/launchos-skill
/plugin install launchos@Jonykit/launchos-skill
```

**通用方式：**
将本目录放到 AI 客户端的 skills 路径下。首次使用时 AI 会自动安装依赖。

最低要求：macOS + LaunchOS 已安装。

## 工作原理

本工具直接读写 LaunchOS 的本地 SQLite 数据库（`~/Library/Application Support/LaunchOS/LaunchOS.sqlite`）。

**分类逻辑由 AI 动态完成**，而非写死的规则引擎。AI 会：
1. 读取你的全量应用和分组数据
2. 分析应用分布，推理你的使用画像
3. 基于画像 + 应用名 + Bundle ID 推理归属，生成个性化方案
4. 尊重你原有的分组命名习惯
5. 展示方案等你确认后才执行

每次使用前会自动检查已安装的 LaunchOS 版本是否匹配。版本不一致时会提示风险，由你决定是否继续。
